Automatic Line Following Robot

Arduino-based autonomous robot designed for real-time line tracking and motor control

October 2024 - January 2025 Cynthia Hsiao

I designed and built an autonomous line-following robot using an Arduino microcontroller for real-time control and decision-making. The robot used sensor feedback to detect and follow a path, allowing it to adjust its movement automatically as it travelled along the line.

I integrated an H-bridge motor driver to precisely regulate motor speed and direction, enabling smooth maneuverability and accurate path tracking. This allowed the robot to respond to changing sensor readings and make directional corrections during operation.

To improve sensing performance, I modeled and fabricated a custom five-sensor mounting bracket in SolidWorks to ensure consistent alignment and responsiveness. This helped maintain reliable sensor positioning and improved the robot’s ability to detect the line accurately.

I assembled the circuitry using precise soldering techniques and systematically debugged both the hardware and Arduino code to achieve reliable, consistent operation.

Languages and Utilities Used:

  • CAD (SolidWorks)
  • Arduino Uno
  • Arduino IDE
  • Embedded C++
  • H-Bridge Motor Driver
  • Soldering and Circuit Assembly
